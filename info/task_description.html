You are the modern man who prefer 24-hours time format. But in some places is used 12-hours format. Your task is to convert time from 12-h format into 24-h using the next rules:
- the output format should be 'hh:mm'
- if the output hours is less than 10 - write a '0' before it. For example: '09:05'
You can find some useful information about 12-hours format.

Input: Time in 12-hours format (as string).

Output: Time in 24-hours format (as string).
